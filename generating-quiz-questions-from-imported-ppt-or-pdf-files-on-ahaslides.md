# Generating Slides from Imported Files on AhaSlides

AhaSlides offers a seamless feature that lets users effortlessly turn their documents, images, PowerPoint (PPT), or PDF files into interactive quizzes with our AI-powered tool. This functionality not only saves time but also enhances engagement during training sessions. Here's how it works:

## Click on the Import Button:

- Locate and click the **'Import'** button on the **My Presentation** dashboard

![Import button on My Presentations dashboard](https://help.ahaslides.com/galleryDocuments/edbsnff4e41bca9e23b9a332be80fb775a8dafebe8b1ede949aba378ff2df8cdcc124c43ca47be650b7279e9a8d874d55caa6?inline=true)

- Or in the editing screen.

![Import button in editing screen](https://help.ahaslides.com/galleryDocuments/edbsn585e70c6fc166febe861976ee40ef5a25342e326dd5d149a91327dc112d67a9ec609516668abd87df676e554f31682d1?inline=true)

## Upload and Generate:

Click the upload area to select a file, or drag the file you want to import into it.

![Upload area for file import](https://help.ahaslides.com/galleryDocuments/edbsn9712169b14731120bf92fad1d8b0e1d46a33445e02f59d3eeba5a79e7e4a3b1b35206314df3d128f994393e1ea5eb6a3?inline=true)

## Complete Import:

After selecting your file, you can choose what you want to do with it.

For PPT, PDF, and image files, there will be 3 options:

- **Import and generate slides with AI**
- **Generate interactive slides**
- **Import slides**

![Import options for PPT/PDF/image files](https://help.ahaslides.com/galleryDocuments/edbsn267a2f1e86d1045b5005f8676a7298e28af95626a3cf3491cc9cef6b18b236566df647fa0df823ae40b776675e99cda7?inline=true)

For document files, the only option is **Generate interactive slides**

![Import option for document files](https://help.ahaslides.com/galleryDocuments/edbsncc81eec962cb00fc4fd106304205e00dc25489d0191e613a8fa7506bc6b52c1c45066337e99145e80f6cc9e093985143?inline=true)

After selecting an option, click **Start** to complete the import.

If you pick **Import and generate slides with AI** or **Generate interactive slides**, the AI will read the file and create slides based on its contents.

![AI generating slides from imported file](https://help.ahaslides.com/galleryDocuments/edbsne783848257fe130dc1046d024eea0fc69096d55001b04d868b4d2f62db38399f2fc995bcc9f4ba889d07643aa74b37a9?inline=true)

## Benefits of Using AhaSlides for Quiz Creation

- **Time Efficiency:** Quickly transform existing presentations into interactive quizzes.
- **Enhanced Engagement:** Interactive quizzes make training sessions more engaging.
- **Versatility:** Supports many file formats.
