You can import any PowerPoint or PDF file directly into AhaSlides and present your existing content alongside interactive slides. After importing, you can place AhaSlides quiz, poll, or Q&A slides between your imported slides to add audience interaction at any point in your presentation.

<Note>
  You can import any PPT, PPTX, or PDF file up to **50 MB** and **100 slides**. Larger files may take longer to fully import.
</Note>

---

## Where to find the Import button

There are two places to import a file:

<CardGroup cols={2}>
  <Card title="From the dashboard" icon="house">
    On the **My Presentations** dashboard, click the **Import** button. A pop-up window lets you browse for your file or drag and drop it.
  </Card>
  <Card title="From the editor" icon="pen-to-square">
    Inside the presentation editor, click the **Import** button. Imported slides are added to the end of your existing presentation.
  </Card>
</CardGroup>

---

## Import options

After selecting your file, choose how you want to use it. For PPT, PPTX, PDF, and image files, three options are available:

<Tabs>
  <Tab title="Import and generate with AI">
    **Import and generate slides with AI** imports your file as static slides and then uses AI to suggest and insert interactive slides after relevant sections. This is the best choice when you want to keep your original slides and automatically add engagement points throughout.
  </Tab>
  <Tab title="Generate interactive slides">
    **Generate interactive slides** uses AI to read your file and create a set of interactive slides based on the content — without importing the original slides. Choose this option when you want a fully interactive presentation built from your file.
  </Tab>
  <Tab title="Import slides only">
    **Import slides** brings your file into AhaSlides exactly as it is, with no AI generation. You can always use AI to generate interactive slides later, or add AhaSlides interactive slides manually between the imported ones.
  </Tab>
</Tabs>

<Note>
  For document files (such as Word documents), only **Generate interactive slides** is available.
</Note>

Once you have chosen an option, click **Start** to complete the import.

---

## What happens to an imported file

Each slide from your file is converted into a separate **Content slide** in AhaSlides. These Content slides contain static images of your original slides.

<Warning>
  Animations, embedded audio, and hyperlinks from the original file are **not** imported. Each imported slide is a static image only.
</Warning>

Even though imported slides are static images, they still have full Content slide functionality — you can add text, images, GIFs, and emojis on top of them.

Each imported slide is named with its slide number and the name of the file you uploaded. You can rename them in the **Content** tab of each slide. Your audience cannot see slide names.

---

## Making an imported presentation interactive

Imported slides themselves cannot be made interactive, apart from the default emoji reactions. To add audience interaction to an imported presentation, you have two options:

<Steps>
  <Step title="Option 1 — Use AI generation">
    When importing, choose **Import and generate slides with AI** or **Generate interactive slides**. AI reads your file and inserts interactive slides automatically.
  </Step>
  <Step title="Option 2 — Add slides manually">
    After importing, use the slide panel to insert AhaSlides interactive slides (polls, quizzes, Q&A, word clouds, and more) between your imported slides at any position you choose.
  </Step>
</Steps>

## Frequently asked questions

<Accordion title="Which file formats can I import?">
  AhaSlides accepts **PPT**, **PPTX**, and **PDF** files. You can also import image files and document files, though document files only support the Generate interactive slides option.
</Accordion>

<Accordion title="What are the file size and slide limits?">
  You can import files up to **50 MB** and **100 slides**. Files within these limits may still take a few moments to process, depending on their size.
</Accordion>

<Accordion title="What happens to animations and audio from my PowerPoint?">
  Animations, embedded audio, and links are not carried over. Each slide is imported as a static image on a Content slide.
</Accordion>

<Accordion title="Can I add more slides to an imported presentation?">
  Yes. After importing, you can add any AhaSlides slide type anywhere in the presentation from the slide panel in the editor.
</Accordion>

<Accordion title="Do I need a paid plan to import files?">
  No. Importing files is available on all plans, including Free.
</Accordion>
