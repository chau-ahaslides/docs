The Presentations Dashboard is your home in AhaSlides. Everything — creating, organising, searching, and sharing presentations — starts here. Every time you log in, the dashboard is the first screen you see, giving you immediate access to all your content and the tools to manage it.

## The Three Views

The left sidebar lets you switch between three views of your content.

| View | What it contains |
|---|---|
| **My Presentations** | All presentations you have created or own, organised into any folders you have set up |
| **Shared with Me** | Presentations that other AhaSlides users have shared with you |
| **Trash** | Presentations you have recently deleted — these can be restored before permanent deletion |

## Creating a New Presentation

<Steps>
  <Step title="Click New Presentation">
    Select the **New Presentation** button in the top-left area of the dashboard.
  </Step>
  <Step title="Give it a name">
    Type a name for your presentation in the dialog that appears, then confirm.
  </Step>
  <Step title="Start adding slides">
    The slide editor opens automatically. Add your first slide by choosing an interactive type from the panel on the left — for example a poll, quiz question, or word cloud.
  </Step>
</Steps>

Prefer to work from existing content? You can also import a PowerPoint or PDF directly from the dashboard. See the [Importing a Presentation](#importing-a-presentation) section below for details.

## Organising into Folders

Folders help you keep large libraries tidy — by team, topic, event, or any structure that suits your workflow.

<Steps>
  <Step title="Create a folder">
    Click **New Folder** in the dashboard toolbar and type a name for the folder.
  </Step>
  <Step title="Add presentations to the folder">
    Drag a presentation card into the folder, or open the **⋯** menu on the presentation card, choose **Move to**, and select the destination folder.
  </Step>
  <Step title="Nest folders if needed">
    Open any folder and repeat the process inside it to create sub-folders. AhaSlides supports up to **10 folder levels deep** and up to **200 presentations per folder**.
  </Step>
</Steps>

## Actions on a Presentation

Every presentation card has a **⋯** menu that reveals a set of actions you can take.

| Action | Description |
|---|---|
| **Present** | Launch the presentation in presenter mode immediately |
| **Rename** | Change the display name of the presentation |
| **View Report** | Open the results report showing audience responses from all past sessions |
| **Reset Results** | Clear all stored audience response data so the presentation is ready for a fresh session |
| **Duplicate** | Create an identical copy of the presentation in My Presentations |
| **Share** | Invite collaborators to view or edit the presentation, or copy a sharing link |
| **Move to** | Move the presentation into a folder or back to the root level |
| **Move to Trash** | Send the presentation to the Trash (can be restored before permanent deletion) |
| **Leave** | Remove yourself from a presentation that another user shared with you |

## Bulk Actions

When you need to move or remove several presentations at once, bulk actions save time.

Select multiple presentation cards by clicking the checkbox that appears in the top-left corner of each card when you hover over it. Once two or more cards are selected, a bulk action toolbar appears at the bottom of the screen. From there, choose **Move to** to relocate them all to a folder, or **Move to Trash** to delete them in one step.

## Sorting and Searching

Keep your dashboard organised by controlling how presentations are listed.

- **Sort order** — Click the sort dropdown and choose **Recently Edited**, **Recently Created**, or **Alphabetical** to reorder the cards in the current view.
- **Search** — Use the search bar at the top of the dashboard to find any presentation by name. The search covers all folders and the Shared with Me view simultaneously, so you do not need to navigate into each folder manually.

## Importing a Presentation

If you have an existing slide deck, you can bring it into AhaSlides and add interactive elements on top.

Click **Import** in the dashboard toolbar, upload a **.pptx**, **.ppt**, or **.pdf** file, and AhaSlides converts it into an editable presentation. Each slide in your file becomes a slide in AhaSlides, ready for you to layer on polls, quizzes, and other interactive types.

For full details on supported formats, file size limits, and conversion tips, see the [Importing Files](/features/importing-files) guide.
