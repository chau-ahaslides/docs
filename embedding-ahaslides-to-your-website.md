Find out how to embed the quiz/poll to your website.

To embed AhaSlides, use this iframe code on your website:

`<iframe src="https://audience.ahaslides.com/" style="width: 100%; height:800px"> <p>Your browser does not support iframes.</p> </iframe>`

You can change the link to your own presentation link, or change the size of the audience frame.

Please note that this embedding feature is not officially supported so there might be unexpected bugs.
